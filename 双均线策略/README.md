# 贵州茅台股票双均线策略回测系统

## 项目概述

本项目是一个基于Python的股票策略回测系统，专门用于分析贵州茅台(600519)股票的历史表现。系统使用双均线策略进行回测，支持获取真实的前复权历史数据，并生成详细的回测报告和可视化图表。

## 功能特点

- **真实数据获取**：使用akshare库获取贵州茅台的真实历史数据
- **前复权处理**：自动获取前复权数据，考虑分红、送股等因素
- **双均线策略**：基于短期和长期移动平均线的交叉信号进行交易
- **完整回测**：模拟交易过程，计算交易成本和绩效指标
- **绩效分析**：计算夏普比率、最大回撤等关键绩效指标
- **可视化输出**：生成股价与均线走势图和策略净值曲线
- **健壮性设计**：包含重试机制和错误处理，确保系统稳定运行

## 策略说明

### 双均线策略

1. **原理**：当短期均线上穿长期均线时买入，当短期均线下穿长期均线时卖出

2. **参数设置**：
   - 短期均线：10日均线
   - 长期均线：30日均线

3. **交易规则**：
   - 买入信号：短期均线上穿长期均线
   - 卖出信号：短期均线下穿长期均线
   - 每次交易使用全部资金/持仓

4. **成本计算**：
   - 交易成本率：0.1%（包括手续费和滑点）

## 注意事项

1. **数据获取**：
   - 系统依赖akshare库获取真实数据
   - 若网络连接失败，会自动使用模拟数据作为备份
   - 建议在网络稳定的环境下运行，以获取真实数据

2. **参数调整**：
   - 可以修改`short_window`和`long_window`参数调整均线周期
   - 可以修改`transaction_cost_rate`参数调整交易成本

3. **性能优化**：
   - 系统已添加重试机制，提高数据获取成功率
   - 对于大量数据的回测，可能需要较长时间

4. **结果解释**：
   - 回测结果仅供参考，不构成投资建议
   - 实际交易中还需要考虑更多因素，如市场流动性、滑点等

5. **扩展功能**：
   - 可以扩展其他技术指标和交易策略
   - 可以添加更多绩效评估指标
   - 可以支持更多股票的回测
