# 贵州茅台双均线策略优化版 (program2.py)

## 项目概述

本项目是一个基于双均线策略的股票交易系统，专门针对贵州茅台(600519)股票进行回测和分析。该系统在传统双均线策略的基础上，增加了参数优化、信号增强和风险管理功能，以提高策略的表现。

## 功能特点

### 1. 数据获取
- 使用akshare API获取贵州茅台真实历史数据
- 支持前复权数据，确保分析准确性
- 包含完善的错误处理和重试机制
- 当API调用失败时，自动切换到模拟数据作为备份

### 2. 技术指标
- 双均线指标（可优化参数）
- MACD指标（用于信号增强）
- RSI指标（用于信号过滤）

### 3. 策略优化
- 自动测试不同均线组合，找到最佳参数
- 支持自定义参数范围
- 基于夏普比率评估策略表现

### 4. 风险管理
- 8%止损机制
- 20%止盈机制
- 交易成本计算

### 5. 可视化
- 股价与均线走势
- 交易信号标记
- MACD指标展示
- 策略净值曲线

## 使用方法

### 基本使用
1. 确保虚拟环境已激活
2. 运行程序：
   ```bash
   python program2.py
   ```
3. 程序将自动：
   - 获取贵州茅台真实历史数据
   - 优化策略参数
   - 运行回测
   - 生成可视化结果

### 自定义参数

你可以修改`program2.py`中的以下参数：

- **数据时间范围**：修改`get_data`方法中的`start_date`和`end_date`参数
- **参数优化范围**：修改`optimize_parameters`方法中的`short_windows`和`long_windows`列表
- **风险管理参数**：修改`backtest`方法中的`stop_loss`和`take_profit`参数
- **交易成本**：修改`backtest`方法中的`transaction_cost_rate`参数

## 策略说明

### 双均线策略

1. **基本原理**：当短期均线上穿长期均线时买入，当短期均线下穿长期均线时卖出
2. **信号增强**：结合MACD指标确认趋势方向
3. **信号过滤**：使用RSI指标避免在超买区域买入

### 参数优化

程序会自动测试不同的短期和长期均线组合，基于夏普比率选择最佳参数。默认测试的参数范围：
- 短期均线：5, 10, 15, 20
- 长期均线：20, 30, 40, 50

### 风险管理

- **止损**：当股价下跌超过8%时自动卖出
- **止盈**：当股价上涨超过20%时自动卖出
- **交易成本**：考虑0.1%的交易成本（包括手续费和滑点）

## 输出结果

### 控制台输出

程序运行后，会在控制台输出以下信息：

1. 股价与均线走势
2. 买入/卖出信号标记
3. MACD指标
4. 策略净值曲线

## 注意事项

1. **数据获取**：程序依赖akshare API获取真实数据，可能会受到网络连接和API限制的影响
2. **参数优化**：参数优化会测试多个参数组合，可能需要较长时间运行
3. **回测结果**：回测结果仅供参考，不构成投资建议
4. **交易成本**：实际交易成本可能与假设值不同
